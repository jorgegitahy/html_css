Italico / Ênfase

Texto marcado

Podemos criar um destaque no nosso texto usando a tag MARK.

Texto deletado / riscado

Podemos tambem marcar o texto usando a tag STRIKE

Sublinhado

Para sublinar um texto utilizamos a tag INS.

Subrescrito / Subrescrito

Quando queremos diminuir uma letra ou numero na parte de cima de uma palavra ou formula usamos a tag SUP por exemplo: X20+ Y

Já para diminuir na parte debaixo usamos a tag SUB por exemplo: H20
